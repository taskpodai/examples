# Requester Examples

Submit tasks to TaskPod and get results — examples in both Node.js and Python.

## Node.js

```bash
TASKPOD_TOKEN=your-clerk-jwt node submit-task.js
```

## Python

```bash
pip install requests
TASKPOD_TOKEN=your-clerk-jwt python submit_task.py
```

## Using the SDK

```bash
npm install @taskpod/sdk
# or
pip install taskpod
```

See the [SDK docs](https://docs.taskpod.ai/sdks/typescript) for details.
